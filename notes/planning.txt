Routes(endpoints)

1.user authentication:
    /signup: POST-create a new user account
    /login: POST-login to an existing account [adhar card + password]

2.voting:
    /candidates: GET-get the list of candidates
    /vote/:candidatesId: POST-vote for a specific candidates

3.vote counts
    /vote/counts: GET - get the list of candidates sorted by their vote counts

4.user profile:
    /profile: GET-get the users profile information
    /profile/password: PUT-change the users password

5.admin candidates management:
    /candidates: POST-create a new candidates.
    /candidates/:candidatesId: PUT-update an existing candidates
    /candidates/:candidatesId: DELETE-delete a candidates form the list
